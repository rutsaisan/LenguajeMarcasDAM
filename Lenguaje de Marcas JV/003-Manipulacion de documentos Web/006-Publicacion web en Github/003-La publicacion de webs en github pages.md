GitHub Pages e sun servicio de publicación web gratuito en GitHub

Requerimientos:
1.- Solo se puede usar HTML, CSS, Javascript, JSON (no Python,PHP, MySQL...)
2.- Requiere que el repositorio este abierto
3.- Requiere que la web inicial se llame index.html

Como pueblocar una web en GitHub Pages:

1.- Vamos a GitHub.com
2.- O iniciamos sesión o creamos un usuario
3.- Vamos a repositorios
4.- Creamos un nuevo reposotorio
5.- Ponermos un nombre al repositorio que no haya sido pillado antes
6.- La visiblidad tiene que ser pública
7.- Recomiendo añadir un readme.md
8.- O bien lo editamos en la nube
9.- O bien lo clonamos en nuetsro ordenador con GitHub Desktop
    - File - Clone Repositori 
    - Seleccionamos en nombre del repositorio que hemos creado
    -Ahora el repositorio existe en la carpeta que hemos clonado

10.- Abro la carpeta con un editor y creo un archivo index.html
11.- Le pongo algo de contenido
12.- Si ya he puesto algo de contenido, hago Commit + Push
13.- De vuelta en GitHub.com, accedo a settings (arriba a la derecha)
14.- Menú de la izquierda me voy a pages
15.- En branch esta en "None", elijo "main"
16.- No olvides pulsar el botón Save para guardar

...
17.- Espero entre 5 y 10 minutos
...
